# 💍 Terra Média | Portfólio Front-end

Uma aplicação web imersiva dedicada ao universo de J.R.R. Tolkien, desenvolvida como projeto pessoal para aprofundar e demonstrar conhecimentos em desenvolvimento Front-end moderno.

## Sobre o Projeto

O projeto "Terra Média" é um site responsivo e semântico que reúne informações, imagens, vídeos e poemas sobre a saga O Senhor dos Anéis. A aplicação foi construída do zero, focando na transição de práticas antigas de marcação para os padrões modernos do HTML5 e na criação de layouts adaptáveis.

Este projeto demonstra:
* **Estruturação Semântica:** Uso correto de tags HTML5 (`<header>`, `<main>`, `<article>`, `<figure>`) visando acessibilidade e SEO.
* **Design Responsivo:** Implementação de CSS Grid e Flexbox para garantir que a interface se adapte fluidamente desde telas de celulares até monitores ultrawide.
* **Boas Práticas de CSS:** Uso de variáveis globais (`:root`) para padronização de cores e tipografia temática.

## Tecnologias Utilizadas

* **HTML5:** Estrutura e semântica web.
* **CSS3:** Estilização, variáveis, CSS Grid e Flexbox.
